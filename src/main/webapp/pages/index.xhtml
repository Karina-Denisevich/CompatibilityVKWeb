Совместимость пользователей ВКонтакте

Данное приложение позволит узнать совместимость нескольких пользователей.


В приложении "Совместимость пользователей" сравнение проводится по следующим параметрам:

  1. Знак зодиака.
  2. Жанры аудиозаписей.

В поле для ввода id необходимо ввести id пользователя, который можно скопировать из адресной строки:

Для наиболее точного результата необходимо, чтобы у пользователей, участвовавших в проверке совместимости был открыт доступ к аужиозаписям и дате рождения. Если в результатах в каком-либо поле содержится символ "-", это означает, что пользователь скрыл свои данные в данной области.
